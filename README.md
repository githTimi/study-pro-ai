# StudyPro AI 📚

A mobile learning assistant that transforms study materials into summaries, quizzes, and flashcards.

## Features
- AI-powered summaries (mocked for demo)
- Interactive quiz system
- Flashcard generation
- Personal library (Firestore)
- Authentication (Firebase Auth)

## Tech Stack
- Flutter
- Firebase (Auth + Firestore + Functions)
- OpenAI (architecture implemented, mocked for demo)

## Screens
- Home
- Summary Results
- Quiz Interface
- Library
- Profile
